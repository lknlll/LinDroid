##### 时间复杂度O(n)
假设n代表算法的入参变量，O(f(n))代表算法执行时间与n的函数关系

###### 常数阶O(1)
没有循环都是；

###### 线性O(n)
```
for(i=1; i<=n; ++i){
   j = i;
   j++;
}
```
代码时间即为for循环次数，一层for循环也就是循环n此循环内的代码。

###### 对数阶O(logN)
```

int i = 1;
while(i<n)
{
    i = i * 2;
}
```

每循环一次，i 多乘以2，1\*2\*2\*2\*...\*n个2，也就是2的n次方，执行的次数也就是以2为底n的对数。
